# A BootStrap4 Theme for Hexo

很简单的一个主题，适合新手学习使用。

## 安装使用
```
cd yourblog
git clone https://github.com/kinggozhang/hexo-theme-abst.git  abst
```
修改_config.xml 的主题
```
## Themes: https://hexo.io/themes/
theme: abst
```
## 注意
archive.ejs使用了hexo-additional-helper插件
```
cd yourblog
npm install hexo-additional-helper --save
```
